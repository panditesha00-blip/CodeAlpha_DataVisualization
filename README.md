# CodeAlpha Data Visualization

## 📌 Project Description
This project was completed as part of the CodeAlpha Data Analytics Internship.

The project analyzes a Student Performance dataset and visualizes the data using different charts in Python.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- VS Code

## 📊 Visualizations Created
- Bar Chart
- Line Chart
- Pie Chart
- Histogram
